Think of this as the LSADP (LinuxSA Documentation Project). The hope is to have a few short useful tips available here for a few common situations that don't seem to be concisely documented elsewhere.

We have a grand total of eleven tips so far. Anyone else who would like to write a document for submission here is encouraged to!

  • Disk Partitioning (Alex Garner)

  • Command Pipelines (David Newall)

  • IP Masquerading (David Newall)

  • Finding the Right Manual Page (David Newall)

  • Superblocks (David Newall)

  • Zombies (David Newall)

  • IO Redirection (Richard Sharpe)

  • Command Line Tricks (Richard Sharpe)

  • File Permissions (David Lloyd)

  • Linux, Clocks, and Time (Geoffrey D. Bennett)

  • Using floppy disks and CDROMs (Geoffrey D. Bennett)
