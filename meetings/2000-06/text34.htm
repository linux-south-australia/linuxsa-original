Mounting a filesystem

  • mount -T ext2 /dev/hda1 /mnt

  • No files below mount point can be opened

    • But already open files are fine

  • Mount points and symbolic links

    • Use relative symbolic links
      • No: ln -s /usr/bin/fred /usr/local/fred
      • Yes: cd /usr/local; ln -s ../../bin/fred fred
