Tuning caching

  • Use lots of memory

  • Lot's of data can be in memory

    • So umount can take a long time, especailly with floppies
    • If the memory fails, cached data is lost and files will be inconsistent (even if filesystem is consistent)

  • UPS

    • Opportunity to shut down machine nicely
    • Too many people have UPS but no control cable to computer!
