Linux IP Settings - IP Address, Subnetmask, Default Gateway

Once you have your ethernet card installed, and being "seen" by Linux, you now need to configure the Internet Protocol settings, to allow your Linux box to communicate with other TCP/IP hosts.

There are a minimum of two, and usually three configuration options which need to be set to allow your Linux box to communicate with other TCP/IP hosts. These are :

  • IP Address - This address uniquely identifies the Linux box on the internetwork.

  • Subnetmask - The subnetmask specifies where the boundary between the network/subnet portion and host portion occurs within the IP address.

  • Default Gateway - This is the IP address of a "default" router to use if the IP address of the destination machine is not on the local network (eg. ethernet segment).
