Getting Linux Connected, using Ethernet and TCP/IP

Configuring these settings the easyway - Using Redhat

Unfortunately, I am using an older slackware distribution of Linux, not a Red Hat distribution, so I don't know much about how to configure the IP settings under Red Hat.

However, I do know that you use the graphical system tool under Red Hat to assign the IP address, subnet mask and gateway. The system tool is fairly intuitive.

It would be best to refer to the Red Hat documentation to determine how to configure the IP address, subnet mask and default gateway.
