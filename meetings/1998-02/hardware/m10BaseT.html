Getting Linux Connected, using Ethernet and TCP/IP

A Multi-Node Connection - Connecting using 10BaseT

To connect more than two nodes to a 10BaseT network you do not use a 10BaseT Crossover cable. Doing so could waste a lot of time, as the node using the 10BaseT Crossover Cable just will not work.

To connect together more than 2 nodes in a 10BaseT network, you will need the following components :

  • An ethernet card with a 10BaseT (RJ-45) connector installed in each Linux box to be attached.

  • A 10BaseT cable (wired straight through) for each node to be connected. These cables are quite common in pre-made lengths, and are available at Dick Smith's (again they are pricey), and other places such as Lawrence & Hanson. Data Cabling and Cable Assembly companies should also be able to provide these cables in a number of pre-made and customised lengths.

  • A 10BaseT hub, with enough available ports for the number of Linux boxes being connected. A 10BaseT hub is an electronic device which takes in a signal from a 10BaseT node, and repeats it out onto all the other ports on the hub (another name for a hub is a multi-port repeater). They come in a variety of configurations, ranging from 4 ports through to some configurations with 48 ports. Also, hubs can be joined together to create a larger number of ports. Hubs are available from computer networking equipment suppliers.

Although this method is significantly more expensive than all the other methods of connecting ethernet nodes together, it has become the preferred method of connecting ethernet networks together in the corporate world for the following reasons :

  • You can physically remove a node from the network (ie. the cable and the Linux box) without disrupting the operation of the network.
  • The failure of a cable only affects the node it is attached to. When diagnosing a fault in the network, this saves a lot of time and money.

The only problem with a 10BaseT network like this is the "eggs in one basket" problem. If the hub fails, all nodes will not be able to communicate. This is a good reason to buy a well-known, well-established brand of hub.
