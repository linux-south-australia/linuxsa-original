A Multi-Node Connection - Connecting using 10Base2

Adding additonal nodes to a 10Base2 network is simple. You will need the following components :

  • Another Linux box with an Ethernet card installed, with a 10Base2 interface.

  • An additional segment of RG-58 coaxial cable, with BNC connectors on both ends.

  • An additional T-Piece (which should have been supplied with the Ethernet card).

To attach the node, you remove the terminator from the end of the cable (make sure no one is using the network!), attach the new coax cable segment and T-Piece, and then re-attach the terminator to the end of the ethernet segment. Then plug the T-Piece into the back of the Linux box, and you are ready to go.
