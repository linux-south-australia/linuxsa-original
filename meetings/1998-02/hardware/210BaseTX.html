Getting Linux Connected, using Ethernet and TCP/IP

A 2 Node Connection - 10BaseT Crossover Cable

You will need the following components to connect the two Linux machines together using a 10BaseT Crossover Cable :

  • 2 Ethernet network cards with a 10BaseT Interface.

  • Custom made 10BaseT Crossover cable, with the RJ-45 male connectors wired as follows :

    • 1 (Tx+) --------- 3 (Rx+)
    • 2 (Tx-) --------- 6 (Rx-)
    • 3 (Rx+) --------- 1 (Tx+)
    • 6 (Rx-) --------- 2 (Tx-)

    The remaining pins should either be wired straight through or not wired at all.

    It is a good idea to have this cable made of a different colour outer sheath than traditional blue or grey and labeled as a 10BaseT Crossover if possible. This will save confusing it with a standard 10BaseT cable.

The advantages of this method are :

  • Only 1 cable needed, if you loose it, you have lost everything !
  • If you are using two machines which have 100Mbps Ethernet cards, you can network at 100Mbps instead of 10Mbps.

The disadvantages of this method are :

  • You have to have a customised cable made up. Cable Assembly and Data Cabling companies should be able to do this.
  • You can only connect two nodes together using this method. You need different cables, and additional equipment to connect together more machines using 10BaseT.
