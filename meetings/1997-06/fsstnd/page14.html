The /var Hierachy (cont.)
      /var/adm
        • Obsolete, link to /var/log or remove

      /var/catman
        • Locally formatted manual pages

      /var/lib
        • Application state files for /usr/lib files

      /var/local
        • Variable storage for /usr/local

      /var/lock
        • Various lock files

      /var/log
        • System log files
