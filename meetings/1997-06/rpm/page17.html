Verifying Packages
      rpm -V pkg1 ...

      Verifies:
        • Owner
        • Group
        • Mode
        • MD5 checksum
        • Size
        • Major/minor numbers
        • Symbolic link string
        • Modification time

      Verifying all packages:
        • rpm -Va

      Verifying the package that owns a file:
        • rpm -Vf /bin/bash

      Verifying against a package file:
        • rpm -Vp eject-1.4-3.i386.rpm
