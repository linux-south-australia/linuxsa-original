Selecting Packages
      One installed package:
        • The package name/version/label
          rpm -q libc
          rpm -q libc-5.3.12
          rpm -q libc-5.3.12-18
        • Case sensitive
        • No wildcards

      All installed packages:
        • rpm -qa
        • Use "less" or "grep"

      Query the package owning a file:
        • rpm -qf /bin/ls
