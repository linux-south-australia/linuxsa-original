LILO Configuration
      Sample /etc/lilo.conf: 
      boot=/dev/hda
map=/boot/map
install=/boot/boot.b
prompt
timeout=50
other=/dev/hda1
        label=dos
        table=/dev/hda
image=/boot/vmlinuz
        label=linux
        root=/dev/hda2
        read-only
      "image=" section can be repeated as required to boot different kernels

      You must rerun /sbin/lilo when any changes to this file or any kernels are made
