    Clean up from the inside

  • Find out how people can connect

    • netstat -a !!!
    • rpcinfo -p localhost !!!

  • Check file permissions

    • setuid only when necessary
    • Check that protected data is protected

  • Know thy enemy - know the state of installed software

    • Keep up to date with patches/updates
    • Monitor security mailing lists for alerts !!!
    • Restrict packages installed to only those needed

  • Use some scanning packages

    • COPS
    [ < ^ > ]

    Clean up from the outside

  • Improve security of your site by breaking into it

    • Become a script-kiddie
    • Multiple checklists step-throughs available on the Internet

  • Use some scanning packages

    • SATAN
    • ISS
    [ < ^ > ]

    Add defence in depth

  • Protect each part of the system

    • BIOS, LILO and kernel passwords, single user mode
    • Helps with physical security

  • Apply cryptography where appropriate

    • PGP encrypt files
    • Cryptographic File System

  • Add network protection at various levels

    • ip filtering (ipfw)
    • TCP wrappers
    [ < ^ > ]

    Monitor your System

  • Know what to trust

  • Message logs

    • syslog is typically used by a lot of programs
    • Check files pointed to by /etc/syslog.conf !!!

  • Login history

    • Who logged in from where at what time
    • /var/log/wtmp and last
    [ < ^ > ]
